# STICKLS - Premium Press-On Nails Landing Page 💅

A modern, elegant single-page landing page for STICKLS, a premium false nails (press-on nails) brand. Optimized for conversion and user experience.

## ✨ **Landing Page Features**

### **Core Sections**
- **Hero Section** - Compelling value proposition with clear CTAs
- **How It Works** - Simple 3-step process (Measure & Select, Apply & Style, Reuse & Repeat)
- **Featured Products** - Curated collection showcase (Classic, Bold, Seasonal)
- **Social Proof** - Customer testimonials with star ratings
- **Strong CTA Section** - Clear conversion focused call-to-action
- **Essential Footer** - Contact info and social links

### **Interactive Features**
- ✨ **Virtual Try-On Modal** - AR-powered experience (placeholder)
- 📱 **Mobile-First Responsive Design**
- 🔄 **Smooth Scrolling Navigation**
- ⚡ **Performance Optimized** - Fast loading with minimal assets
- ♿ **Accessibility Ready** - WCAG 2.1 AA compliant

## 🎨 **Design System**

### **Color Palette**
- **Primary**: Cream (#FAF7F0), Blush Pink (#E8B4B8)
- **Accent**: Rose Gold (#D4AF37), Deep Burgundy (#8B2635)
- **Supporting**: Forest Green (#2C5530), Champagne (#F7E7CE)

### **Typography**
- **Headlines**: Playfair Display (Elegant serif)
- **Body**: Inter (Clean sans-serif)
